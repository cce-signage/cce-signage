GitHub Access

Create one at GitHub → Settings → Developer settings → Fine-grained tokens. Set Repository access to only cce-signage/cce-signage and grant a single permission: Contents → Read and write. Tokens expire (max 1 year) — when publishing starts failing with 401, create a new one. A remembered token is never shown in this field again — the placeholder just confirms one is saved.
