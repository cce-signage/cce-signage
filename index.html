Edit Weekly Agenda

Add, edit, or remove events below, then select Save. The sign arranges everything automatically by day and start time, so the order you enter rows here does not matter — Sort by Day & Time just tidies this list. Use + Space to show an event in more than one room and Duplicate to copy a line. Set a start and end time for each session; the sign always displays them in 24-hour format. To open this editor later, add ?edit=1 to the sign URL or press Ctrl + Shift + E.

The hosted sign reads its content from config.json in the GitHub repository and re-checks it every 5 minutes, so config.json overrides anything saved here. To update every sign, make your changes, select Download config.json, then upload that file to the GitHub repository replacing the existing config.json.
